# 🏥 Proyecto Semana 1: Ficha de Presentación de Dominio

> **🎯 ÚNICO ENTREGABLE**: Este proyecto corresponde a la ficha de presentación del dominio asignado.

---

## 📋 Dominio Asignado

**Dominio**: Farmacia y Salud

---

## 🎯 Objetivo

Construir un script en JavaScript que se ejecute en Node.js y muestre en consola una ficha organizada sobre una entidad del dominio Farmacia y Salud.

Se aplican conceptos básicos como:
- Uso de `console.log()`
- Tipos de datos (string, number, boolean)
- Expresiones aritméticas
- Comentarios en el código

---

## 📋 Descripción

El script representa un "carnet digital" de un medicamento dentro de una farmacia.

Se muestra información relevante como:
- Nombre del medicamento
- Código identificador
- Categoría
- Estado
- Precio
- Disponibilidad
- Información general del dominio

---

## ✅ Requisitos Cumplidos

- ✔ Uso de `console.log()` en todo el script
- ✔ Uso de tipos de datos: string, number y boolean
- ✔ Inclusión de una operación matemática (cálculo de descuento)
- ✔ Mínimo 3 comentarios en español
- ✔ Código organizado con separadores visuales
- ✔ Script ejecutable sin errores
