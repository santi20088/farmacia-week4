# 🏥 Semana 2: Variables y Tipos de Datos (Farmacia y Salud)

## 🎯 Objetivos de Aprendizaje

Al finalizar esta semana, serás capaz de:

- ✅ Declarar variables con `const` y `let` aplicadas a datos de farmacia
- ✅ Explicar por qué `var` está obsoleto y no debe usarse
- ✅ Usar tipos primitivos en el contexto de medicamentos (`string`, `number`, `boolean`, `null`, `undefined`)
- ✅ Identificar tipos de datos con `typeof` en información de salud
- ✅ Convertir tipos con `Number()`, `String()` y `Boolean()`
- ✅ Aplicar nomenclatura correcta (camelCase) en variables del sistema
- ✅ Usar numeric separators para precios y cantidades (`1_000_000`)
